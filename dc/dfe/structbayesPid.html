◆ ProbType

enum bayesPid::ProbType : int

Table of the most probable particle type.

Types of probabilities that can be computed N.B. the order is important, detectors first!

Enumerator
kTOF 
kTPC 

Probabilities with the TOF detector.

kNDet 

Probabilities with the TPC detector.

kPrior 

Number of detectors.

kMerged 

Prior probabilities.

kBayesian 

Merged probabilities without the prior probability applied (not normalized)

kNProb 

Bayesian probability with the prior probability applied (normalized)
